# College Appointment System - UnQue Assignment
Backend APIs for a college appointment system for a college appointment system

## Setup
##### 1. Clone the repository
```bash
git clone https://github.com/tej-as-kamble/UnqueCloudbookAssignment.git
cd UnqueCloudbookAssignment
```

##### 2. Install dependencies
```bash
npm install
```

##### 3. Create .env file
```bash
PORT=3000
MONGO_URI=mongodb://127.0.0.1:27017/unque-assignment
JWT_SECRET=jwt_secret_key
```
##### 4. Running Tests
```bash
npm test
```

##### 5. Start the server
```bash
npm start
```
